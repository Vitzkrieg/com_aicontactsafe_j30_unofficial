*Esta não é uma versão oficial - é apenas uma atualização que foi feita para torná-lo compatível com o Joomla 3 Não nos responsabilizamos por eventuais bugs, contudo podem sim reportar os mesmos em nossa página de contato ou, de preferência na página do Git Hub *

Meu site com esta extensão: http://pontomega.com.br/downloads/ai-contact-safe-para-joomla-3-nao-oficial#download

Endereço original da extensão: http://www.algisinfo.com/en/home-bottom/37-aicontactsafe-extension.html

No pacote do componente você vai encontrar:

  • mod_aiContactSafe : um módulo para exibir um formulário de contato em qualquer posição do teu tema
  • plg_aiContactSafeForm : um plugin para colocar o contato em um artigo
  • plg_aiContactSafeLink: um plugin para colocar um link de uma página de contato em um artigo Você pode instalar e/ou atualizar qualquer um deles a partir do Painel de Controle do aiContactSafe e ver mais informações sobre todos eles nos artigos de algisinfo.com.

Algumas das características do aiContactSafe são:

  • CAPTCHA com a possibilidade de especificar palavras de código para cada idioma ou seqüências aleatórias, facilmente integrado em seu projeto por configurações de cores e de fundo na cor a ser usada
  • Perfis ( múltiplos formulários, com diferentes campos personalizados )
  • campos personalizados ilimitados de diferentes tipos ( Textbox, Checkbox, Checkbox list, Combobox, Radio list, Editbox, Date, Email, Email list, Hidden, Separator, Joomla contact, File )
  • adicionar um prefixo ou sufixo para o campo
  • redirecionar a página especificada quando a mensagem é enviada com sucesso!
  • qualquer número de anexos de trabalho, com ou sem AJAX
  • suporte a SEF por meio ARTIO JoomSEF ou sh404SEF, ou você pode usar o SEF nativo do Joomla
  • suporte a diversos idiomas (arquivos de idioma e Joomfish)
  • arquivos de idioma para Inglês, árabe, búlgaro, checo, dinamarquês, alemão, grego, húngaro, espanhol, francês, italiano, holandês, polaco, Brazilian Portuguese, Portuguese, Romanian, Russian, Slovak, Serbian (Cyrillic), Swedish, Turkish, Ukrainian, Lithuanian ( some not completed )
  • especificar as informações de contato com um editor WYSIWYG
  • instruções detalhadas para cada página de administração em Inglês (e algumas das traduções mencionadas acima)
  • bloquear usuários por IP
  • bloquear mensagens que contenham palavras de uma lista que você pode definir
  • proibir automaticamente IP que envia mensagens com palavras bloqueadas em um período de tempo especificado
  • gravar as mensagens no banco de dados com o IP a partir do qual eles foram enviados (pode ser desativado a partir do Painel de Controle)
  • capacidades de configuração estendida
  • estrutura MVC
  • temas de email
  • ordenar os campos para cada perfil

Algumas das características do aiContactSafe são:

  • CAPTCHA with the possibility to specify code words for each language or random strings, easily integrated in your design by settings the colors and background color to use
  • profiles ( multiple forms with different custom fields )
  • unlimited custom fields of different types ( Textbox, Checkbox, Checkbox list, Combobox, Radio list, Editbox, Date, Email, Email list, Hidden, Separator, Joomla contact, File )
  • add a prefix or suffix to the field
  • redirect to specified page when the message is successfully sent
  • any number of attachments working with or without AJAX
  • SEF support through ARTIO JoomSEF or sh404SEF, or you can use the Joomla's native SEF
  • multilingual support ( language files & Joom!Fish )
  • language files for English, Arabic, Bulgarian, Czech, Danish, German, Greek, Spanish, French, Italian, Dutch, Polish, Brazilian Portuguese, Romanian, Russian, Slovak, Serbian (Cyrillic), Swedish, Turkish, Ukrainian ( some not completed )
  • specify contact information with a WYSIWYG editor
  • detailed instructions for each administration page in English ( and some of the translation mentioned above )
  • block users by IP
  • block messages that contain words from a list you can define
  • automatically ban IP that sends messages with blocked words in a specified period of time
  • record the messages in the database with the IP from which they've been sent ( can be disabled from Control Panel )
  • extended configuration capabilities
  • MVC architecture
  • email templates
  • order the fields for each profile
